# NA8US EXPANSION - Final Bento Architecture (v4.3)

## 1. ПРАВИЛО ZERO TOLERANCE (КРИТИЧНО)
- ПОЛНОСТЬЮ УДАЛИТЬ блоки: "ГЕРМАНИЯ", "СИСТЕМА: АКТИВНА", "LOG_001". Их не должно быть в коде.
- Использовать только 3 типа плиток: БИО, СОЦСЕТИ, СТАТЬЯ.

## 2. ВИЗУАЛ: TRUE BENTO & DYNAMICS
- **Стиль:** Карточки с "левитацией". Каждая плитка имеет `box-shadow: 0 10px 30px rgba(0,0,0,0.5)`.
- **Параллакс:** Реализовать через библиотеку Vanilla-tilt.js (пусть ИИ подключит её через CDN). Карточки должны наклоняться за мышью.
- **Фон:** Глубокий черный #0a0a12. Рамки плиток — неоновый градиент (Cyan to Purple).

## 3. КОНТЕНТ И ССЫЛКИ (СТРОГО РУССКИЙ)
- **Блок "БИО" (2x2):** Текст про ИИ-интеграцию.
- **Блок "СЕТЬ" (2x1):** 6 иконок (Instagram, Telegram, YouTube, TikTok, LinkedIn, GitHub). Ссылки из прошлых версий.
- **Блок "СТАТЬЯ" (2x2):** Картинка logs/articles/article1/article1.jpg. Заголовок: "Твой клиент больше не человек...".

## 4. ИСПРАВЛЕНИЕ ОШИБОК
