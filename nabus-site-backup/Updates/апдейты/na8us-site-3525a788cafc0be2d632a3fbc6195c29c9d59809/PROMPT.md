# NA8US EXPANSION - Bento Design (v5.0)
- **Concept:** Чистый Bento Grid (3-4 крупные карточки).
- **Style:** Темный режим, скругление углов 32px, тонкая неоновая рамка (Cyan).
- **Layout:** 1. Блок BIO (большой, слева).
  2. Блок СТАТЬЯ (с картинкой на весь фон, справа).
  3. Блок СОЦСЕТИ (длинная узкая плитка снизу под ними).
- **Interactions:** Карточки должны плавно «дышать» (параллакс при движении мыши).
